
## 1. Introduction to MySQL

MySQL is an open-source Relational Database Management System (RDBMS)
that stores data in tables using rows and columns. It uses SQL
(Structured Query Language) to manage and retrieve data.

### Key Features

-   Open-source and free\
-   Fast and reliable\
-   Supports large datasets\
-   Works perfectly with PHP\
-   Uses SQL for data operations

### Where MySQL Is Used?

-   Websites and Web Apps\
-   WordPress, Drupal\
-   E-commerce platforms\
-   School/College management systems\
-   Banking/Finance applications

------------------------------------------------------------------------

## 2. Creating Database & Operations in MySQL

### 2.1 Creating a Database

``` sql
CREATE DATABASE school;
```

### Select a Database

``` sql
USE school;
```

------------------------------------------------------------------------

## 2.2 Creating a Table

``` sql
CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    class VARCHAR(20),
    marks INT
);
```

------------------------------------------------------------------------

## 2.3 Insert Records

``` sql
INSERT INTO students (name, class, marks)
VALUES ('Amit', '12th', 88);
```

------------------------------------------------------------------------

## 2.4 Select/Read Data

``` sql
SELECT * FROM students;
```

Filtered data example:

``` sql
SELECT name, marks FROM students WHERE marks > 80;
```

------------------------------------------------------------------------

## 2.5 Update Data

``` sql
UPDATE students 
SET marks = 95 
WHERE id = 1;
```

------------------------------------------------------------------------

## 2.6 Delete Data

``` sql
DELETE FROM students WHERE id = 1;
```

------------------------------------------------------------------------

## 2.7 Drop a Table

``` sql
DROP TABLE students;
```

------------------------------------------------------------------------

## 2.8 Drop a Database

``` sql
DROP DATABASE school;
```

# 3. Querying a MySQL Database with PHP

PHP connects with MySQL using:

-   **MySQLi**
-   **PDO (PHP Data Objects)**

------------------------------------------------------------------------

# 3.1 Using MySQLi (OOP Method)

### Connect to MySQL

``` php
<?php
$conn = new mysqli("localhost", "root", "", "school");

if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

echo "Connected!";
?>
```

------------------------------------------------------------------------

### Insert Data Using MySQLi

``` php
$sql = "INSERT INTO students (name, class, marks)
        VALUES ('Rohit', '10th', 76)";

if ($conn->query($sql) === TRUE) {
    echo "Record inserted!";
} else {
    echo "Error: " . $conn->error;
}
```

------------------------------------------------------------------------

### Fetch Data Using MySQLi

``` php
$sql = "SELECT * FROM students";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while ($row = $result->fetch_assoc()) {
        echo $row['name'] . " - " . $row['marks'] . "<br>";
    }
}
```

------------------------------------------------------------------------

# 3.2 Using PDO (Recommended & Secure)

### Connect Using PDO

``` php
<?php
$dsn = "mysql:host=localhost;dbname=school";
$user = "root";
$pass = "";

try {
    $pdo = new PDO($dsn, $user, $pass);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connected!";
} catch (PDOException $e) {
    echo "Connection failed: " . $e->getMessage();
}
?>
```

------------------------------------------------------------------------

### Insert Data Using PDO (Prepared Statements)

``` php
$sql = "INSERT INTO students (name, class, marks) 
        VALUES (:name, :class, :marks)";

$stmt = $pdo->prepare($sql);

$stmt->execute([
    ':name' => 'Surya',
    ':class' => '12th',
    ':marks' => 91
]);

echo "Data Inserted!";
```

------------------------------------------------------------------------

### Fetch Data Using PDO

``` php
$stmt = $pdo->prepare("SELECT * FROM students");
$stmt->execute();
$data = $stmt->fetchAll(PDO::FETCH_ASSOC);

foreach ($data as $row) {
    echo $row['name'] . " - " . $row['marks'] . "<br>";
}
```

------------------------------------------------------------------------

# Summary

-   MySQL is a relational database widely used in web development.\
-   SQL helps create, insert, update, delete, and read data.\
-   PHP interacts with MySQL using **PDO** (secure) or **MySQLi**.\
-   PDO is recommended for secure and scalable applications.
