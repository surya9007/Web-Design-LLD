
## ‚≠ê 1. Using PHP to Access a Database

PHP provides built‚Äëin extensions to connect and interact with databases.
The most commonly used are:

-   **PDO (PHP Data Objects)**
-   **MySQLi (MySQL Improved)**

### üîπ Steps in PHP Database Connectivity

1.  **Create a connection** to the database.\
2.  **Send SQL commands** (SELECT, INSERT, UPDATE, DELETE).\
3.  **Fetch the results** from the database (if any).\
4.  **Close the connection** after work is done.

### üîπ Example Flow (Simple Theory)

When PHP connects to a MySQL database: - It uses a **hostname** (like
`localhost`) - A **username** (like `root`) - A **password** - A
**database name**

If the connection is successful ‚Üí PHP executes queries.\
If failed ‚Üí PHP throws an error or exception.

## ‚≠ê 2. Relational Databases and SQL

### üîπ Relational Database (RDBMS)

A relational database stores information in **tables**.\
Each table has: - **Rows** ‚Üí individual records - **Columns** ‚Üí data
fields (name, email, age, etc.)

Examples of RDBMS: - MySQL - PostgreSQL - Oracle - MariaDB

### üîπ SQL (Structured Query Language)

SQL is used to: - Create tables - Insert data - Update data - Delete
data - Retrieve data

### üîπ Common SQL Commands 

#### ‚úÖ Create a Table

``` sql
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(150)
);
```

#### ‚úÖ Insert Data

``` sql
INSERT INTO users (name, email) 
VALUES ('John Doe', 'john@example.com');
```

#### ‚úÖ Read Data

``` sql
SELECT * FROM users;
```

#### ‚úÖ Update Data

``` sql
UPDATE users SET name='Alex' WHERE id=1;
```

#### ‚úÖ Delete Data

``` sql
DELETE FROM users WHERE id=1;
```

## ‚≠ê 3. PHP Data Objects (PDO)

### üîπ What is PDO?

PDO is a **universal database connection layer** in PHP.\
It works with many databases: - MySQL - SQLite - PostgreSQL - Oracle -
SQL Server

### üîπ Advantages

-   More secure (supports **prepared statements**)
-   Flexible (change DB type easily)
-   Object‚Äëoriented (clean code)

------------------------------------------------------------------------

### üîπ Example: PDO Connection

``` php
<?php
$dsn = "mysql:host=localhost;dbname=testdb";
$username = "root";
$password = "";

try {
    $pdo = new PDO($dsn, $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connected successfully!";
} catch (PDOException $e) {
    echo "Connection failed: " . $e->getMessage();
}
?>
```

------------------------------------------------------------------------

### üîπ Example: Fetch Data Using PDO

``` php
<?php
$stmt = $pdo->prepare("SELECT * FROM users");
$stmt->execute();
$data = $stmt->fetchAll(PDO::FETCH_ASSOC);

foreach ($data as $row) {
    echo $row['name'] . " - " . $row['email'] . "<br>";
}
?>
```

------------------------------------------------------------------------

### üîπ Example: Insert Using Prepared Statement (Secure)

``` php
<?php
$sql = "INSERT INTO users (name, email) VALUES (:name, :email)";
$stmt = $pdo->prepare($sql);

$stmt->execute([
    ':name' => "Surya",
    ':email' => "surya@example.com"
]);

echo "Data inserted!";
?>
```

## ‚≠ê 4. MySQLi Object-Oriented Interface

### üîπ What is MySQLi?

MySQLi is an improved version of the old MySQL extension.\
Supports: - **Object-oriented style** - **Procedural style**

Works **only with MySQL**.

------------------------------------------------------------------------

### üîπ MySQLi Connection Example (OOP Style)

``` php
<?php
$conn = new mysqli("localhost", "root", "", "testdb");

if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

echo "Connected successfully!";
?>
```

------------------------------------------------------------------------

### üîπ MySQLi Select Query Example

``` php
<?php
$sql = "SELECT * FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while ($row = $result->fetch_assoc()) {
        echo $row['name'] . " - " . $row['email'] . "<br>";
    }
}
?>
```

------------------------------------------------------------------------

### üîπ MySQLi Insert Example

``` php
<?php
$sql = "INSERT INTO users (name, email) VALUES ('Rahul', 'rahul@example.com')";

if ($conn->query($sql) === TRUE) {
    echo "New record created!";
} else {
    echo "Error: " . $conn->error;
}
?>
```

## ‚≠ê 5. PDO vs MySQLi (Detailed Comparison)

  Feature               PDO               MySQLi
  --------------------- ----------------- ------------------------------
  Database Support      ‚úî Multiple        ‚ùå Only MySQL
  Prepared Statements   ‚úî Yes             ‚úî Yes
  API Style             Object‚Äëoriented   Object‚Äëoriented + Procedural
  Error Handling        Exceptions        Basic Error Messages
  Flexibility           High              Limited
  Security              Very Strong       Strong

## ‚≠ê 6. When to Use What?

### ‚úî Use **PDO** if:

-   You may change the database later.
-   You want maximum security.
-   You prefer object‚Äëoriented coding.

### ‚úî Use **MySQLi** if:

-   You are working **only with MySQL**.
-   You want a simpler approach.
-   You want both procedural and OOP styles.
